### HIGH-1: Incremental Backup Engine

**Objective:** Replace current `pg_dump`/`mysqldump` full-only approach with proper incremental backup support using pgBackRest and Percona XtraBackup.

**Current state:** `BackupType` field supports `"full"`/`"incremental"`, but `executeDump()` ignores it — always runs full dump. Need real incremental.

**Key decisions:**
- pgBackRest for PostgreSQL (WAL-based PITR) — server-level, not per-DB
- XtraBackup for MySQL, Mariabackup for MariaDB — page-level incremental
- Docker image needs pgBackRest, XtraBackup, Mariabackup binaries
- Schedule modal already has "Full"/"Incremental" selector — just need backend

#### Task 1.2: Add incremental execution to backup service

**Files:**
- Modify: `internal/backup/service.go` — add `pgBackRestBackup()`, `xtraBackupBackup()` methods
- Modify: `internal/backup/service.go` — update `executeDump()` to branch on `backupType`

**Logic:**
```go
func (s *Service) executeDump(conn *connection.Connection, dbName, backupType string) ([]byte, error) {
    switch conn.DBType {
    case "postgresql":
        if backupType == "incremental" {
            return s.pgBackRestBackup(conn, dbName)
        }
        return s.pgDump(conn, dbName)
    case "mysql", "mariadb":
        if backupType == "incremental" {
            return s.xtraBackupBackup(conn, dbName)
        }
        return s.mySQLDump(conn, dbName, backupType)
    default:
        return nil, fmt.Errorf("unsupported database type: %s", conn.DBType)
    }
}
```

**pgBackRest:**
- Need to manage pgbackrest.conf for each PG connection
- Stanza per server, backup type maps to pgbackrest `--type=full|incr|diff`
- Output: directory of WAL archives → tar → gzip → S3

**XtraBackup:**
- `xtrabackup --backup --target-dir=<dir>` for full
- `xtrabackup --backup --target-dir=<dir> --incremental-basedir=<prev_dir>` for incremental
- Stream to stdout: `xbstream` format
- Need to manage target dirs for incremental chain

func (s *Service) compressData(data []byte, method string) []byte {
    switch method {
    case "zstd":
        // Use github.com/klauspost/compress/zstd
        return zstdCompress(data)
    case "none":
        return data
    default: // "gzip"
        return gzipCompress(data)
    }
}
